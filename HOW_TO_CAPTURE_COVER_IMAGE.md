## 🚀 Quick Start (3 Steps)

### Step 1: Open the HTML File
```bash
# Navigate to the ML folder
cd ML

# Open the HTML file in your browser
# Option 1: Double-click the file
# Option 2: Right-click → Open with → Chrome/Edge/Firefox
```

### Step 2: Take a Screenshot
Use one of these methods:

#### Method A: Browser Screenshot (Recommended)
1. Open the HTML file in **Chrome** or **Edge**
2. Press `F12` to open Developer Tools
3. Press `Ctrl+Shift+P` (Windows) or `Cmd+Shift+P` (Mac)
4. Type "screenshot" and select **"Capture screenshot"**
5. The image will be saved to your Downloads folder

#### Method B: Windows Snipping Tool
1. Open the HTML file in full screen
2. Press `Windows + Shift + S`
3. Select the entire cover area
4. Save the screenshot

#### Method C: Mac Screenshot
1. Open the HTML file in full screen
2. Press `Cmd + Shift + 4`
3. Select the entire cover area
4. The screenshot will be saved to your Desktop

### Step 3: Use the Image
The captured image is now ready to use as your mobile app cover!

## 🎯 Screenshot Tips

### For Best Quality
1. **Use Chrome or Edge** - Best screenshot quality
2. **Full screen mode** - Press `F11` before capturing
3. **Zoom to 100%** - Press `Ctrl+0` to reset zoom
4. **Wait for animations** - Let particles settle for 2-3 seconds
5. **Capture full area** - Include all edges

### Browser Screenshot Settings
1. Open Developer Tools (`F12`)
2. Press `Ctrl+Shift+P` (Windows) or `Cmd+Shift+P` (Mac)
3. Type "screenshot"
4. Choose:
   - **"Capture screenshot"** - Visible area only
   - **"Capture full size screenshot"** - Entire page (recommended)

## 🖼️ File Formats

### Recommended Formats
- **PNG** - Best quality, larger file size (recommended for app stores)
- **JPG** - Good quality, smaller file size (recommended for web)
- **WebP** - Best compression, modern browsers only

### Compression Tools
- **TinyPNG** - https://tinypng.com/ (online)
- **Squoosh** - https://squoosh.app/ (online)
- **ImageOptim** - https://imageoptim.com/ (Mac)
- **RIOT** - https://riot-optimizer.com/ (Windows)
