# E-Commerce BI Platform

## Overview
This project was developed as part of the **PABI – 4ERPBI Program** at **Esprit School of Engineering** (Academic Year 2025–2026).  
It consists of an e-commerce platform integrated with a Business Intelligence pipeline, allowing teams to analyze sales data, automate ETL processes, and visualize insights.

## Features
- Collect data from multiple sources (SQL databases, CSV/Excel files, ERP systems, archived PDFs)  
- ETL processing using Talend  
- Data storage in SQL Server, MySQL, PostgreSQL, and a Data Warehouse  
- BI dashboards using Power BI  
- Frontend built with Angular  
- Backend implemented with Flask  
- Workflow orchestration with Apache Airflow  
- Automation via N8N  

## Tech Stack

### Frontend
- Angular

### Backend
- Flask (Python)
- Talend (ETL)
- MySQL, SQL Server, PostgreSQL (Data Storage)
- Apache Airflow (Orchestration)
- N8N (Automation)
- Power BI (BI Visualization)

## Architecture
- Data Sources → ETL (Talend) → Data Storage (SQL/Database/Warehouse) → Backend APIs (Flask) → Frontend (Angular) → BI Dashboards (Power BI)  
- Automation and orchestration handled by N8N and Apache Airflow
